# WeatherSky - Demo Instructions

## How to Run the Application

1. **Start the server**:
   ```bash
   cd ProjectWeather
   npx serve .
   ```
   Or use any local server like Live Server in VS Code.

2. **Open your browser** and navigate to `http://localhost:3000` (or the port shown by your server).

## How to Use

1. **Select a Country**: Choose from the dropdown with flag emojis
2. **Search for City**: Type to filter cities in real-time
3. **View Weather**: Click any city to see detailed forecast
4. **Add Favorites**: Save cities for quick access
5. **Browse History**: See your recent searches
